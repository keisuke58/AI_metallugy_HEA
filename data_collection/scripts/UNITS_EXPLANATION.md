# 単位についての説明（RMSE/MAEがGPaなのはなぜ？）

## 📊 なぜRMSE/MAEがGPa単位なのか

### 1. 予測対象（ターゲット変数）の単位

このプロジェクトでは、**弾性率（Elastic Modulus / Young's Modulus）**を予測しています。

- **物理量**: 弾性率（Young's Modulus）
- **単位**: **GPa（ギガパスカル）**
- **データセット内の列名**: `elastic_modulus`

### 2. データの実際の値

データファイル（`data_with_features.csv`）を見ると：

```csv
alloy_name,elastic_modulus,source,...
Al0.5CoCrCuFeNi,161.1,DOE/OSTI,...
Al0.3CoCrFeNi,168.0,DOE/OSTI,...
Al0.5CrCuFeNi2,172.1,DOE/OSTI,...
CoCrFeNi,173.9,DOE/OSTI,...
```

これらの値（161.1, 168.0, 172.1, 173.9など）は**GPa単位**で記録されています。

### 3. 誤差指標の単位

**RMSE（Root Mean Squared Error）**と**MAE（Mean Absolute Error）**は、以下のように計算されます：

#### RMSEの計算式
```
RMSE = √(1/n × Σ(予測値 - 実際の値)²)
```

#### MAEの計算式
```
MAE = 1/n × Σ|予測値 - 実際の値|
```

### 4. なぜGPa単位になるのか

誤差指標は**予測値と実際の値の差**を計算するため、**ターゲット変数と同じ単位**になります。

- **実際の値**: 161.1 GPa
- **予測値**: 例えば 180.0 GPa
- **誤差**: 180.0 - 161.1 = **18.9 GPa**

したがって、RMSEやMAEも**GPa単位**で表現されます。

### 5. 具体例

Transformerモデルの結果：
- **Test RMSE**: 42.52 GPa
  - これは「平均的に、予測値と実際の値の差が約42.52 GPa」という意味
- **Test MAE**: 28.35 GPa
  - これは「平均的に、予測値と実際の値の絶対差が約28.35 GPa」という意味

### 6. データセットの統計

データセット全体の弾性率の範囲：
- **最小値**: 2.73 GPa
- **最大値**: 621.33 GPa
- **平均**: 139.26 GPa
- **中央値**: 122.61 GPa

実験データ（322サンプル）の場合：
- **範囲**: 約36-330 GPa
- **平均**: 約162 GPa

### 7. まとめ

| 項目 | 説明 |
|------|------|
| **予測対象** | 弾性率（Elastic Modulus） |
| **単位** | GPa（ギガパスカル） |
| **RMSE/MAEの単位** | GPa（ターゲット変数と同じ） |
| **理由** | 誤差 = 予測値 - 実際の値（同じ単位同士の引き算） |

## 📚 参考：GPaとは

**GPa（ギガパスカル）** = 10⁹ パスカル = 1,000,000,000 パスカル

弾性率の一般的な範囲：
- **ゴム**: 0.001-0.1 GPa
- **プラスチック**: 0.1-10 GPa
- **金属**: 50-400 GPa
- **セラミックス**: 100-500 GPa
- **ダイヤモンド**: 1000+ GPa

高エントロピー合金（HEA）の弾性率は、通常**50-300 GPa**の範囲にあります。
