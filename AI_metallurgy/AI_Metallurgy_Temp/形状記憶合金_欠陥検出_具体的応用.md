# 形状記憶合金（SMA）への欠陥検出：CFRP手法の具体的応用

## 📋 概要

CFRP研究で開発した**FEA + GNN手法**を形状記憶合金（Shape Memory Alloys, SMA）の欠陥検出に応用する具体的な方法を説明します。

## 🔬 形状記憶合金の特徴と欠陥の種類

### 形状記憶合金の基本特性

1. **相変態（Phase Transformation）**
   - マルテンサイト相（低温、変形可能）
   - オーステナイト相（高温、元の形状に戻る）
   - 温度依存性

2. **形状記憶効果（Shape Memory Effect）**
   - 変形後、加熱により元の形状に回復

3. **超弾性（Superelasticity）**
   - 大きな変形を可逆的に回復

### 形状記憶合金における欠陥の種類

#### 1. **相変態関連の欠陥**
- **不完全な相変態**: 一部の領域が相変態しない
- **不均一な相変態**: 相変態が不均一に進行
- **相変態の阻害**: 欠陥により相変態が阻害される

#### 2. **結晶欠陥**
- **転位（Dislocations）**: 結晶構造の不完全性
- **粒界（Grain Boundaries）**: 結晶粒の境界
- **双晶（Twinning）**: マルテンサイト相の双晶構造

#### 3. **製造欠陥**
- **気孔（Porosity）**: 製造過程での気泡
- **介在物（Inclusions）**: 不純物の混入
- **組成不均一性**: 合金組成のばらつき

#### 4. **疲労損傷**
- **き裂（Cracks）**: 繰り返し負荷によるき裂
- **相変態疲労**: 相変態の繰り返しによる損傷

```python
def extract_node_features(fea_result):
    """
    形状記憶合金に特化した特徴量抽出
    """
    features = []
    
    for element in fea_result.elements:
        # 基本力学量
        stress = element.stress_tensor  # [σ_xx, σ_yy, σ_xy]
        strain = element.strain_tensor   # [ε_xx, ε_yy, ε_xy]
        
        # 相変態関連
        phase_fraction = element.martensite_fraction  # マルテンサイト相の割合
        transformation_strain = element.transformation_strain  # 相変態ひずみ
        
        # 温度関連
        temperature = element.temperature
        temperature_gradient = element.temperature_gradient
        
        # 応力不変量
        von_mises_stress = calculate_von_mises_stress(stress)
        principal_stresses = calculate_principal_stresses(stress)
        
        # 特徴量ベクトル
        feature_vector = np.concatenate([
            stress.flatten(),           # 3次元
            strain.flatten(),           # 3次元
            [phase_fraction],           # 1次元
            [transformation_strain],    # 1次元
            [temperature],               # 1次元
            temperature_gradient,       # 2次元（2Dの場合）
            [von_mises_stress],         # 1次元
            principal_stresses          # 2次元
        ])
        
        features.append(feature_vector)
    
    return np.array(features)
```

### ステップ3: GNNモデルの構築

#### 3.1 GAT（Graph Attention Network）モデル

```python
import torch
import torch.nn as nn
from torch_geometric.nn import GATConv, global_mean_pool

class SMA_Defect_Detection_GNN(nn.Module):
    """
    形状記憶合金の欠陥検出用GNNモデル
    CFRP研究で使用したGAT/GATv2アーキテクチャをベース
    """
    def __init__(self, input_dim, hidden_dim, num_classes=19):
        super().__init__()
        
        # GAT層
        self.gat1 = GATConv(
            input_dim, hidden_dim, 
            heads=8, dropout=0.1, concat=True
        )
        self.gat2 = GATConv(
            hidden_dim * 8, hidden_dim, 
            heads=1, dropout=0.1, concat=False
        )
        
        # 分類層
        self.classifier = nn.Sequential(
            nn.Linear(hidden_dim, hidden_dim // 2),
            nn.ReLU(),
            nn.Dropout(0.2),
            nn.Linear(hidden_dim // 2, num_classes)
        )
        
    def forward(self, x, edge_index, batch):
        """
        x: ノード特徴量 [num_nodes, input_dim]
        edge_index: エッジインデックス [2, num_edges]
        batch: バッチ情報
        """
        # GAT層1
        x = self.gat1(x, edge_index)
        x = torch.relu(x)
        x = torch.dropout(x, p=0.1, train=self.training)
        
        # GAT層2
        x = self.gat2(x, edge_index)
        x = torch.relu(x)
        
        # グラフレベルの特徴量（全ノードの平均）
        graph_features = global_mean_pool(x, batch)
        
        # 分類
        output = self.classifier(graph_features)
        
        return output
```

#### 3.2 マルチタスク学習（欠陥検出 + 位置推定）

```python
class SMA_MultiTask_GNN(nn.Module):
    """
    欠陥の種類と位置を同時に予測
    """
    def __init__(self, input_dim, hidden_dim):
        super().__init__()
        
        # 共通のGAT層
        self.gat1 = GATConv(input_dim, hidden_dim, heads=8)
        self.gat2 = GATConv(hidden_dim * 8, hidden_dim, heads=1)
        
        # タスク1: 欠陥分類
        self.defect_classifier = nn.Sequential(
            nn.Linear(hidden_dim, hidden_dim // 2),
            nn.ReLU(),
            nn.Linear(hidden_dim // 2, 19)  # 19クラス分類
        )
        
        # タスク2: 欠陥位置推定（ノードレベル）
        self.location_predictor = nn.Sequential(
            nn.Linear(hidden_dim, hidden_dim // 2),
            nn.ReLU(),
            nn.Linear(hidden_dim // 2, 1)  # 欠陥確率
        )
        
    def forward(self, x, edge_index, batch):
        x = self.gat1(x, edge_index)
        x = torch.relu(x)
        x = self.gat2(x, edge_index)
        x = torch.relu(x)
        
        # グラフレベルの分類
        graph_features = global_mean_pool(x, batch)
        defect_class = self.defect_classifier(graph_features)
        
        # ノードレベルの位置推定
        defect_location = torch.sigmoid(self.location_predictor(x))
        
        return defect_class, defect_location
```

```python
from torch_geometric.data import Dataset

class SMA_Dataset(Dataset):
    """
    形状記憶合金の欠陥検出データセット
    """
    def __init__(self, fea_results, defect_labels, transform=None):
        self.fea_results = fea_results
        self.defect_labels = defect_labels
        self.transform = transform
        
    def __len__(self):
        return len(self.fea_results)
    
    def __getitem__(self, idx):
        # FEA結果を取得
        fea_result = self.fea_results[idx]
        
        # グラフに変換
        graph = fea_mesh_to_graph(fea_result)
        
        # ラベルを取得
        label = self.defect_labels[idx]
        
        if self.transform:
            graph = self.transform(graph)
        
        return graph, label
```

#### 4.2 訓練ループ

```python
def train_model(model, train_loader, val_loader, num_epochs=100):
    """
    モデルの訓練
    """
    optimizer = torch.optim.Adam(model.parameters(), lr=0.001)
    criterion = nn.CrossEntropyLoss()
    
    for epoch in range(num_epochs):
        # 訓練モード
        model.train()
        train_loss = 0
        
        for batch in train_loader:
            graph, labels = batch
            
            # 予測
            predictions = model(graph.x, graph.edge_index, graph.batch)
            
            # 損失計算
            loss = criterion(predictions, labels)
            
            # 逆伝播
            optimizer.zero_grad()
            loss.backward()
            optimizer.step()
            
            train_loss += loss.item()
        
        # 検証
        model.eval()
        val_loss = 0
        val_acc = 0
        
        with torch.no_grad():
            for batch in val_loader:
                graph, labels = batch
                predictions = model(graph.x, graph.edge_index, graph.batch)
                
                loss = criterion(predictions, labels)
                val_loss += loss.item()
                
                # 精度計算
                pred_classes = predictions.argmax(dim=1)
                val_acc += (pred_classes == labels).float().mean().item()
        
        print(f"Epoch {epoch+1}/{num_epochs}")
        print(f"Train Loss: {train_loss/len(train_loader):.4f}")
        print(f"Val Loss: {val_loss/len(val_loader):.4f}")
        print(f"Val Acc: {val_acc/len(val_loader):.4f}")
```

### グラフニューラルネットワーク（GNN）と材料科学

14. **NIST GNN models for defect prediction**:

15. **DefiNet (Defect-Informed Equivariant GNN)**:
    - Explicitly marks defects in crystal graphs, uses equivariant message passing, predicts relaxed atomic structures. Tested on >14,000 defect structures. (2025, Nature Scientific Reports)

    - Predicts vacancy-type defect formation enthalpies from ideal crystal structures. Applied to clean energy materials. (NREL, 2024)

17. **GNN + FEM for CFRP defect detection**:
    - Finite element method simulations generate stress field data, represented as graph over surface nodes. GAT classifies defect locations in 3D within perforated CFRP. Achieved macro F1 ≈ 61% for 19 categories. (2024, DOAJ)

19. **Comprehensive Evaluation of Graph Neural Networks and Physics Informed Learning for Surrogate Modelling of FEA** (2025):
