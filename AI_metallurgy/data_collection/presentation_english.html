Project Overview

Objective

Predict elastic modulus (Young's Modulus) of High Entropy Alloys (HEA) using machine learning

Achievements

  • Collected and integrated 322 data points
  • Developed and evaluated 8 different models
  • Achieved best performance: R² = 0.67
