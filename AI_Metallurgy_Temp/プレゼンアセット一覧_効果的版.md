### スライド2: なぜE（弾性率）に着目したか？

#### アセット1: 弾性率比較図（必須）⭐⭐⭐⭐⭐

**タイプ**: バーグラフまたは比較表

**データ**:
```
材料                    弾性率 (GPa)
─────────────────────────────────
皮質骨                   15-30
海綿骨                   0.02-6
Ti-6Al-4V (従来)        110-120
ステンレス鋼              >200
Co-Cr合金                >200
目標 (現実的)            60-90
目標 (理想的)            30
```

**デザイン**:
- 骨（緑色）、従来インプラント（赤色）、目標（青色）で色分け
- 骨の範囲をハイライト表示

**ツール**: Python (Matplotlib/Seaborn), Excel, PowerPoint

---

#### アセット2: 応力シールディングのメカニズム図（必須）⭐⭐⭐⭐⭐

**タイプ**: 概念図（イラスト）

**内容**:
- 左側: 高弾性率インプラント（赤、太い矢印で負荷を多く負担）
- 右側: 低弾性率インプラント（青、細い矢印で負荷を分散）
- 骨への負荷を色の濃淡で表現

**ツール**: PowerPoint, Canva, Draw.io

---

#### アセット3: 無菌性緩みの原因（推奨）⭐⭐⭐⭐

**タイプ**: 円グラフまたはリスト

**データ**:

```
─────────────────────────────
応力シールディング        60%
感染                     25%
周辺骨折                 15%
```

**ツール**: PowerPoint, Excel

### スライド6: 結果と性能比較

#### アセット8: モデル性能比較バーグラフ（必須）⭐⭐⭐⭐⭐

**タイプ**: バーグラフ

**データ例**:
```
モデル    R²      RMSE (GPa)
─────────────────────────
LIN       0.65    15.2
L         0.68    14.5
R         0.67    14.8
P         0.72    13.2
KNN       0.75    12.5
RF        0.82    10.8
SVR       0.78    11.5
MLFFNN    0.85    9.5
```

**デザイン**:
- R²は高いほど良い（緑色系）
- RMSEは低いほど良い（青色系）
- 最良モデルをハイライト

**ツール**: Python (Matplotlib/Seaborn), Excel

#### アセット9: 予測値 vs 実験値の散布図（必須）⭐⭐⭐⭐⭐

**タイプ**: 散布図

**データ**: 
- X軸: 実験値（GPa）
- Y軸: 予測値（GPa）
- 対角線（y=x）を追加
- 各モデルごとに色分け

**デザイン**:
- 対角線に近いほど良い
- 残差を色の濃淡で表現
- 10 GPaの誤差範囲を表示

**ツール**: Python (Matplotlib/Seaborn), Excel

## 🛠️ アセット作成ツール推奨

### グラフ・図表作成
1. **Python (Matplotlib/Seaborn/Plotly)** ⭐⭐⭐⭐⭐
   - 高品質なグラフ
   - 再現性が高い
   - 無料

2. **Excel** ⭐⭐⭐⭐
   - 簡単
   - プレゼンに直接貼り付け可能

3. **MATLAB** ⭐⭐⭐⭐
   - 3Dプロットに強い
   - 学術的

### 概念図・フローチャート
1. **PowerPoint** ⭐⭐⭐⭐⭐
   - 最も一般的
   - 簡単

2. **Draw.io** ⭐⭐⭐⭐⭐
   - 無料
   - オンライン
   - 高品質

3. **Canva** ⭐⭐⭐⭐
   - 美しいデザイン
   - テンプレート豊富

## 💡 アセット作成のコツ

### 1. **色使い**
- 骨・目標: 緑色系
- 従来インプラント・問題: 赤色系
- 本研究・解決策: 青色系
- 統一感を持たせる

### 2. **フォント**
- タイトル: 18-24pt
- 本文: 14-18pt
- 統一されたフォントを使用

### 3. **データの信頼性**
- 最新研究（2024-2025）のデータを使用
- 出典を明記
- 不確実性を表現（誤差棒など）

### 4. **可読性**
- シンプルに
- 1つの図表に1つのメッセージ
- 過度に複雑にしない
